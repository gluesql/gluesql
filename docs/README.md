# GlueSQL Docs

This website is built with [Zensical](https://zensical.org/).

## Installation

```sh
uv tool install zensical==0.0.43
```

If Zensical is already installed with `uv`, update it with:

```sh
uv tool upgrade zensical
```

## Local Development

```sh
zensical serve
```

## Build

```sh
zensical build
```

The static site is generated into the `build` directory.
